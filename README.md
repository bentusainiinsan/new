# Rental Service - Sonipat Home Service NXT Gen

A comprehensive rental service platform for home services in Sonipat, designed to connect customers with quality service providers.

## Project Structure

```
/client         - Frontend application (React)
/server         - Backend API (Node.js/Express)
/apps-script    - Google Sheets/Email automation scripts
```

## Features

### Core Functionality
- **User Management**: Customer registration and authentication
- **Service Catalog**: Browse and search available home services
- **Booking System**: Schedule and manage service appointments
- **Real-time Availability**: Check service provider availability
- **Payment Integration**: Secure online payment processing
- **Rating & Reviews**: Customer feedback and service ratings

### Admin Features
- **Dashboard**: Overview of bookings, revenue, and analytics
- **Service Management**: Add/edit/remove services and providers
- **Booking Management**: Track and manage all service requests
- **User Management**: Manage customers and service providers
- **Reports**: Generate business insights and reports

### Automation
- **Google Sheets Integration**: Automated data sync and reporting
- **Email Notifications**: Automated booking confirmations and reminders
- **SMS Alerts**: Real-time notifications for bookings and updates

### Additional Features
- **Multi-language Support**: Hindi and English
- **Mobile Responsive**: Optimized for mobile devices
- **Location-based Services**: Find services near you
- **Emergency Services**: Quick access to urgent home services
- **Service History**: Track past bookings and services

## Tech Stack

### Frontend (/client)
- React.js
- Redux for state management
- Material-UI / Tailwind CSS
- Axios for API calls

### Backend (/server)
- Node.js
- Express.js
- MongoDB/PostgreSQL
- JWT Authentication
- Payment Gateway Integration

### Automation (/apps-script)
- Google Apps Script
- Google Sheets API
- Gmail API
- Calendar API

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- MongoDB/PostgreSQL
- Google Cloud Platform account (for Apps Script)
